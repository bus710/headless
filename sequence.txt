# general sequence for a new linux box

## first of all

- git install 
- mkdir ~/repo
- clone the headless and scripts repos under ~/repo

## in bus710/headless

- 01_basic
- 02_go_install
- 03_flutter_install
- 04_nvim_install > plug-ins install (:PlugInstall in nvim)
- 05_docker_install
- sshd config

## in bus710/scripts

- 01_bash_setup
- 05_after_install_basic
- 06_after_install_heavy
- place the dock icons > settings/file/terminal/gitkraken/code/slack/chromium

## etc

- Reduce power off time of cups-browsed (sudo systemctl edit cups-browsed.service > [Service] TimeoutStopSec=10)
- 97_tlp > sudo tlp-stat -p > /etc/default/tlp > CPU_SCALING_MAX_FREQ_ON_AC=2400000
- change terminal color
- change desktop background
- about_gdm3_bgcolor > /usr/share/gnome-shell/theme/gdm3.css > lockDialogGroup > #222222
- about_grub_parameter > /etc/default/grub > nosplash

## in the settings and tweaks GUIs

- about_fcitx > settings > language install > install fcitx
- power, universal access, dock, notification configuration
- caps-lock disable

## app settings

- chromium > gmail sync and github login
- vscode, slack, insync, gitkraken install
- vscode extensions > vim, go, flutter, dracula, markdown-pdf
- vscode go plugin activation
- startup application configuration

## repos

- gows, fltws, gfws, docs
