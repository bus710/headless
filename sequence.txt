# general sequence for a new linux box

## 1. first of all

- git install 
- mkdir ~/repo
- git clone https://github.com/bus710/headless
- git clone https://github.com/bus710/scripts

## 2. bus710/headless

- 01_basic
- 02_golang_install
- 03_flutter_install
- 04_rust_install
- 07_go_flutter_desktop_install
- 08_nvim_install > plug-ins install (:PlugInstall in nvim)
- 09_docker_install
- sshd config

## 3. bus710/scripts

- 01_bash_setup
- 02_basic_packages
- 03_heavy_packages
- 04_tlp_install
- 10_icons
- 11_fcitx_install
- 12_dracula_terminal_theme

## 4. configurations in the terminal

- scripts/04_tlp_install > sudo tlp-stat -p
    - /etc/default/tlp > CPU_SCALING_MAX_FREQ_ON_AC=2400000
- scripts/about_gdm3_bgcolor
    - /usr/share/gnome-shell/theme/gdm3.css > lockDialogGroup > #222222
- scripts/about_grub_parameter
    - /etc/default/grub > nosplash
    - sudo update-grub
- Reduce power off time of cups-browsed 
    - sudo systemctl edit cups-browsed.service
    - [Service] TimeoutStopSec=10

## 5. configurations in the settings/tweaks

- scripts/about_fcitx > settings > language install
- settings > background, power, universal access, dock, notification
- tweaks > caps-lock disable

## 6. app settings

- chrome > gmail sync and github login
- vscode, slack, insync, gitkraken, android studio install
- vscode 
    - extensions > vim, go, flutter, dracula, markdown-pdf, sftp
    - go plugin activation
- edit startup application
- place the dock icons 
    - settings/file/terminal/gitkraken/code/slack/chrome

## 7. repos

- gows, fltws, gfws, docs
