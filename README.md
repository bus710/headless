# Headless

To install these for headless x64 debian/ubuntu servers:
- Global git config
- Basic packages and environment variables
- Zsh and oh-my-zsh
- Go SDK
- Flutter SDK
- NodeJS for nvim
- Nvim (appimage version)
- Docker and Docker composer
- Ufw and some configuration
